### Usage

```bash
rev OPTIONS FILE
```

### Flags

```bash
-V, --version
    Display version information and exit.

-h, --help
    Display help text and exit.
```

### Examples

#### reverse a string

```bash
echo 'string' | rev
```

#### reverse a file

```bash
rev file1
```
